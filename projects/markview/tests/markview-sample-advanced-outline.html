MarkView

MarkView is a Chrome extention for reading markdown file with an outline view (scrollable and expansible), support Github Flavored Markdown which including table styles (sortable) and code block syntax highlight.

Advanced outline: a neat automatic scroll-spying table of contents

Features

  1. Viewing markdown file in HTML format
  2. Auto reload local file when file is changed
  3. Show outline beside the content in scrollable way
  4. Support Github Flavored Markdown table styles and code highlight.
  5. Highlight the code area for programming languages(eg. ```ruby)
